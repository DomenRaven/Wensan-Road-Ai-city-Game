## 四、v1.1 核心决策（摘要）

| 项 | v1.1 结论 |
|----|-----------|
| **Phase 1 栈** | Godot 4.6 + Cursor + godot-mcp + 本地 RAG |
| **推迟** | FastAPI · Redis · React Kiosk → Phase 3 展陈 |
| **MVP 品类** | 横版闯关 · 射击 · 塔防（W1–W2） |
| **AI 约束** | 只改 `config/game_config.json`；禁止改 `core/` |
| **SLA** | 目标 **P90≤10min** + `demo_preset` 硬回退 |
| **Agent 流程** | `query_rag.py` → 改 config → `run_project` → fix ≤2 轮 |
