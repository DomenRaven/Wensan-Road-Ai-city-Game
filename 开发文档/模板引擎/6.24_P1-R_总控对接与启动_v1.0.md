## 六、环境与回归命令

```powershell
# 后端（E2E 前必须常驻，勿用 TestClient 触发 bootstrap 孤儿清理）
.\05-工具脚本\run_backend.ps1

# Kiosk + 教育版
python -m http.server 8080
# 教育入口：http://127.0.0.1:8080/kiosk/edu/

# 配置校验
python 05-工具脚本\validate_creative_templates.py

# HTTP 六款回归（窗8 基线，P1-R 后应仍 6/6）
python 05-工具脚本\e2e_b_edu_batch.py http://127.0.0.1:8000

# platformer 回归（可选）
python 05-工具脚本\e2e_b_edu_platformer.py http://127.0.0.1:8000
```

**Session 池满**：`GET /sessions` → 逐个 `DELETE /sessions/{id}` 后再跑 E2E。

**WinError 32**：`workspace_guard.py` 已吞 PermissionError；后端用 `run_backend.ps1` 常驻。
