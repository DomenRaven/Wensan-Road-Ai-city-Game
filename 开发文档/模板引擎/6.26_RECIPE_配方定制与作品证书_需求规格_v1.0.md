# 6.26 · 配方定制与作品证书 · 需求规格 v1.0

> **起算**：2026-06-26  
> **决策**：用户确认**暂停** [`6.26_SKILLS_补小技能_需求规格_v1.0.md`](./6.26_SKILLS_补小技能_需求规格_v1.0.md) 全线施工，改走 **RECIPE** 三板斧。  
> **目标**：在不改动 `templates/*/core/**` 的前提下，强化「AI 辅助个性化创作」的展厅叙事与可感知差异。

---

## 一、产品一句话

孩子在 B4 填写**手感配方** → AI 写入 `game_config.json` → B5/B6 展示制作过程 → **领取作品登记证书** → B7 试玩体感与左侧代码高亮一致。

**不再**在 B4 承诺「小技能勾选即玩法生效」。

---

## 二、甲方 / 产品目标

| # | 需求 | 方案 | 验收 |
|---|------|------|------|
| R-R1 | 移除 B4「选择小技能」 | **方案 B** | 七款问卷无 `skill_pick`；`enabled_skills` 恒 `[]` |
| R-R2 | 每款再加 1～2 个可感知 tuning 题 | **方案 C** | 仅 `core` 已在读字段；试玩开/关对比明显 |
| R-R3 | 创作完成可下载/打印纪念物 | **方案 A** | B6 作品登记证书：品类·名称·配方·时间 |
| R-R4 | 不破坏已冻结 core 主线 | 硬约束 | `git diff templates/*/core/` 为空 |
| R-R5 | 与 B7 教育高亮一致 | 锚点同步 | 新增题有 `code_anchor_id` |

## 三、范围

### 在范围

- `config/creative_templates/{genre}.json` — 删 `q_skill`、增 tuning 题  
- `config/code_anchors/{genre}.json` — 新锚点（按品类解冻）  
- `kiosk/edu/` — B6 证书 UI、打印/下载  
- `backend` — 仅当证书需服务端字段时（默认前端组装，**不改 analyzer**）  
- 文档库、整合清单、评审记录、工作状态 §八

### 不在范围

- `templates/{genre}/core/**` 任何修改  
- `core/skills/*.gd` 接线、`optional_skills.json` ID 变更  
- LLM 生成玩法逻辑  
- 联机 / 存档 / 内购

### 与 SKILLS 关系

| 项目 | 状态 |
|------|------|
| SKILLS 文档库 | **保留**，标「展后 backlog」 |
| `core/skills/` 预制代码 | 保留，作课后扩展素材，**不在 B4 露出** |
| platformer 已接线技能 | 不拆线；问卷不再勾选即可 |

## 四、分期交付

```text
窗 RECIPE-B     七款问卷：删 skill + 增 tuning + anchors（可单窗或按品类解冻分批）
        ↓
窗 RECIPE-A     B6 作品登记证书（HTML/CSS · 打印 · 下载 PNG 可选）
        ↓
窗 RECIPE-E2E   validate + batch smoke + 用户 2~3 遍创作流
        ↓
经验条目 + 整合清单勾选 + 收工评审
```

**可与 P3-3 Godot 窗口分区并行**；同窗禁止混改 `play.py` 归位逻辑与 RECIPE 大改。

## 五、配方真相源（不变）

| 层级 | 文件 | 职责 |
|------|------|------|
| 问卷 | `config/creative_templates/{genre}.json` | `questions[]` · `tuning_path` · `code_anchor_id` |
| 默认 | `templates/{genre}/config/game_config.json` | 基准值 · `clamp_percent` 参照 |
| 合并 | `backend/.../analyzer.py` | ±30% clamp · 写 workspace `game_config.json` |
| 运行时 | `GameConfig` autoload | `get_tuning()` / `get_theme()` |
| 教育 | `config/code_anchors/{genre}.json` | B7 行号与 caption |
