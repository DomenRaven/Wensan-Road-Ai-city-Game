## 1. 用户故事

> 作为展厅参观者（儿童），我看到明亮干净的蓝白色界面，像学校科技角而不是程序员黑窗口。

---

## 2. 视觉规范

### 2.1 色板（`kiosk_edu_spec.colors` v2）

| Token | 色值 | 用途 |
|-------|------|------|
| `background` | `#f0f7ff` | 页面背景 |
| `surface` | `#ffffff` | 卡片、面板、顶栏底 |
| `code_bg` | `#f8fafc` | 代码区、输入框 |
| `accent` | `#2563eb` | 主按钮、阶段条激活 |
| `accent_light` | `#dbeafe` | 芯片选中、hover |
| `text` | `#0f172a` | 主文字 |
| `muted` | `#64748b` | 副标题、hint |
| `panel_border` | `#bfdbfe` | 分隔线 |
| `highlight` | `#fef08a` | B7 代码高亮（保持） |
| `keyword` | `#1d4ed8` | 代码关键字（浅底上加深） |
| `string` | `#b45309` | 代码字符串 |
| `success` | `#16a34a` | 成功态 |
| `danger` | `#dc2626` | 错误态 |

### 2.2 组件

| 组件 | 要求 |
|------|------|
| 顶栏 `.edu-header` | 白底半透明 + 浅蓝底边；品牌字用 `accent` |
| 主按钮 `.btn-primary` | 蓝底白字；禁用灰 `#94a3b8` |
| 次按钮 `.btn-secondary` | 白底蓝边 |
| 代码区 | 浅灰底 `#f8fafc`；行号 `#94a3b8`；**禁止**纯黑 #000 大面积 |
| 星空 `edu-ambient` | 星云改为浅蓝 `#e0f2fe` 低对比；星点 `#3b82f6` 淡化 |
| 品类动画 `.genre-scene` | 边框 `accent_light`；内底可保留略深游戏预览 |

### 2.3 A 链 `/kiosk/`（同步范围）

| 项 | 要求 |
|----|------|
| 宫格卡片 | 白底 + 蓝描边；选中 `accent_light` |
| 进度条/按钮 | 与 B 链 accent 一致 |
| 实现 | 复用 CSS 变量或从 spec 生成 `kiosk/theme-tokens.css` |

---

## 3. 实现要点

1. `edu-styles.css` 顶部 `:root` 改为读取 spec 默认值（与现逻辑一致，可 JS 注入或手写同步）。
2. 删除/覆盖 v1.0 深色硬编码：`#181825`、`#1e1e2e` 面板底。
3. `code-theater.js` 剧场背景读 `theater.background` → 改为 `#f8fafc`。
4. 对比度：正文与背景对比度 ≥ 4.5:1。

---
