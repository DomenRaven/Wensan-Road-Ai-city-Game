## 1. B2 · 起名

### 1.1 用户故事

> 系统已听懂我想做横版闯关，问我游戏叫什么，我可以自己起名或点推荐名。

### 1.2 界面

| 元素 | 要求 |
|------|------|
| 确认句 | 「我知道了！我们一起制作一个【{品类显示名}】小游戏吧！」 |
| 追问 | 「你希望它叫什么呢？」 |
| 输入框 | max 20 字 · 与 `kiosk_ui_spec` 一致 |
| 推荐芯片 | 该品类 ≥4 个 · 点击填入输入框 |
| 品类图标 | 使用 `assets/previews/{slug}.png` |

### 1.3 推荐名（首期预制）

| slug | 推荐名 |
|------|--------|
| platformer | 星星大冒险、跳跳小子、管道奇遇、金币猎人 |
| shmup | 雷霆小队、星空守护者、小小飞行员、弹幕闪避 |
| survivor | 糖果幸存者、怪物克星、时间大作战、升级小英雄 |
| pingpong | 弹弹乐、乒乓对决、教室球王、快慢球 |
| fighting | 像素拳王、双人对决、校园格斗赛、闪电出拳 |
| parkour | 无尽奔跑、障碍小子、滑铲达人、晨跑冒险 |
| racing | 欢乐赛车、弯道之王、小小车手、冲刺吧 |

来源：`creative_templates.{slug}.name_suggestions` 或 `GET /creative/name-suggestions`。

### 1.4 数据写入

- `meta.display_name`
- `theme.title`（若 theme 块存在）
- 顶栏立即显示作品名

### 1.5 验收

- [ ] 7 品类各有 ≥4 芯片
- [ ] 空名禁止下一步
- [ ] 特殊字符过滤（仅中英文数字空格）

## 2. B3 · 双栏布局

### 2.1 用户故事

> 起好名字后，我看到左边是我的「{作品名}工作区」，右边是即将玩游戏的地方。

### 2.2 布局规格

```
┌─────────────────────────────────────────────────────┐
│ 顶栏：Logo · 作品名 · [创作|制作|试玩] 三段指示      │
├──────────────────────┬──────────────────────────────┤
│ 左 ≥45%              │ 右 Godot 预览区               │
│ ┌ 文件树（伪）        │ ┌────────────────────────┐  │
│ │ 📁 config           │ │                        │  │
│ │ 📁 core             │ │   Godot / 占位插画      │  │
│ └ 代码区 + 行号       │ │   object-fit: contain    │  │
│                      │ └────────────────────────┘  │
└──────────────────────┴──────────────────────────────┘
```

### 2.3 左栏（伪工作区）

| 项 | 要求 |
|----|------|
| 标题 | `{display_name} 工作区` |
| 文件树 | 固定展示 config / core 顶层（不可展开编辑） |
| 代码区 | 只读 `<pre>` · 等宽字体 ≥14px |
| B3 初始内容 | 占位文案：「你的游戏代码会出现在这里」 |
| 滚动 | B3 默认锁定跟手（`touch-action: none` 于代码区） |

### 2.4 右栏（Godot）

| 项 | 要求 |
|----|------|
| B3 状态 | 品类预览图或「即将开始制作」插画 |
| B6 后 | 嵌入试玩 · contain · 参考 racing 540×960 |
| 最小高度 | 不低于左栏 80% |

### 2.5 顶栏阶段指示

| 段 | 对应阶段 |
|----|----------|
| 创作 | B1–B4 |
| 制作 | B5 |
| 试玩 | B6–B7 |

### 2.6 配色（`kiosk_edu_spec`）

- 背景：深色 IDE `#1e1e2e`
- 代码底：`#282a36`
- 高亮预留：`#fef08a`
- 强调色：`#38bdf8`（与现 Kiosk 一致）

### 2.7 验收

- [ ] 1280×720 与 1920×1080 大屏无横向溢出
- [ ] 左栏宽度 ≥45%
- [ ] 作品名顶栏全程可见
- [ ] 触控按钮 ≥48px
