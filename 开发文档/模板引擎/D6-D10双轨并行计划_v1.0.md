## 五、并行约束（Agent / 人工）

| 规则 | 说明 |
|------|------|
| 单会话单模块族 | `kiosk/` **或** `backend/` **或** `templates/{one_genre}/` |
| 禁止改 core | `templates/*/core/` 与 repo 根 `core/` |
| AI 创作只改 config | `workspace/.../config/game_config.json` |
