## 三、各任务技术要点

### P1-04 · 意图词表

| 项 | 要求 |
|----|------|
| 配置 | `config/intent_genre_lexicon.json` 七款 keywords ≥8 · `example_chips` 七条与 E2E 一致 |
| 后端 | `intent.py` 以 lexicon 为唯一真相 · 删除或降级 `DEFAULT_INTENT_KEYWORDS` 重复 |
| 前端 | `b1-intent.js` 示例芯片从 spec/lexicon 读取 · fallback 与 lexicon 同源 |
| 可选 | `GET /intent/lexicon` 只读暴露（非必须） |

**验收**：七句 intent（见附录 A.9）`match-genre` 全对。

---

### P1-05 · B4 通用表单

| widget | 行为 |
|--------|------|
| `single_choice` | 已有 · 保持 option-card |
| `skill_pick` | 多选芯片 · `skill_ids` 来自题目 · `optional:true` 可跳过 · 提交 `string[]` |
| 加载 | **必须** `GET /creative/templates/{genre}` · 禁止仅 platformer fallback |
| 校验 | `validate()` 覆盖 optional 题 |

**必读**：`config/creative_templates/*.json` 各款 `q_skill` 结构。

---

### P1-06 · B2 推荐名

| 项 | 要求 |
|----|------|
| 主路径 | `GET /creative/name-suggestions?genre=` |
| 数据源 | `creative_templates/{slug}.json` → `name_suggestions`（≥4） |
| fallback | `kiosk_edu_spec.json` → `name_suggestions[genre]` 仅 API 失败时 |
| UI | 芯片点击填入 input · 与 platformer 现交互一致 |

---

### P1-08 · play/action 统一桥

| 项 | 要求 |
|----|------|
| API | `POST /sessions/{id}/play/action` body: `{"action_id":"jump"}` |
| 写入 | append `workspace/{id}/.edu_actions.jsonl` · 与 Godot 桥同格式 |
| 校验 | session 存在 · workspace 已 generate · action_id 非空 |
| 测试 | `backend/tests/test_play_action.py` 或并入现有 play 测试 |
| Godot | **本窗可不改** `edu_action_bridge.gd`（文件写入仍有效） |

---

### P1-09 · Kiosk 入口分流

| 入口 | 行为 |
|------|------|
| `/kiosk/edu/` | 保持 · 教育 B 链主入口 |
| `/kiosk/?mode=edu` | 重定向或 `wizard.js` 检测后跳转 edu |
| A 链 | `/kiosk/` 默认 shortcut/七宫格不变 |
| B 链旧 | footer「完整创作向导」不变 |
| UI | header 标明当前模式（可选小标签） |
