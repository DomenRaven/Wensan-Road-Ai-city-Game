## 八、数据流（workspace 生命周期）

```text
POST /sessions
  → Redis/memory session（无 workspace）

B1–B4
  → payload.intent_raw · creative_answers

POST /analyze-requirements
  → payload.analyze_result · code_map_preview

POST /generate/v2
  → copytree templates/{slug} → workspace/{id}/
  → merge game_config.json
  → 可选 copy _edu bridge
  → 刷新 code_map 真实行号

POST /play/launch
  → godot --path workspace/{id}/

B7 试玩
  → .edu_actions.jsonl 追加
  → 前端高亮 code_map[action_id]

POST /release
  → shutil.rmtree workspace/{id}/
```
