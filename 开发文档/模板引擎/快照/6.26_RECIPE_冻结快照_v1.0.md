## 三、解冻流程

1. 用户确认解冻范围（问卷 / 证书 / 代码区 / E2E）
2. 修改后执行：
   ```powershell
   python 05-工具脚本\freeze_recipe_snapshot.py write
   python 05-工具脚本\freeze_recipe_snapshot.py verify
   ```
3. 若改 `creative_templates` 或 `code_anchors`，同步刷新对应品类 freeze：
   ```powershell
   python 05-工具脚本\freeze_platformer_snapshot.py write
   python 05-工具脚本\freeze_genre_snapshot.py write
   # … 其它品类同理
   ```
4. 回归：`validate_creative_templates` · `e2e_b_edu_batch` · browser smoke · `e2e_recipe_a_certificate`
