## 二、排障地图（A 链快玩）

```text
浏览器 /kiosk/
  wizard.js — loadBootstrap · renderS1 · renderS9 · launchGodotPlay
        ↓
FastAPI  POST /sessions → wizard/S1 {data:{genre}} → play/launch
        ↓
godot_launcher — resolve_project_path → templates/{slug}/（无 workspace 时）
        ↓
外置 Godot 窗口（GODOT_PATH）
```

| 症状 | 先查 | 常见根因 |
|------|------|----------|
| S1 空白或 <7 卡 | `config/kiosk_ui_spec.json` featured · `GET /genres` | slug 不在 registry · API 未启 |
| S9 launch 400 | Network：`wizard/S1` body | 缺 `{data:{genre}}` · session 无 genre |
| launch 503 | 响应 detail · 后端日志 | `GODOT_PATH` 空 · 模板缺 `project.godot` · 启动即退出 |
| 路径是 workspace 非 templates | launch 响应 `project_path` | 误走 full 流且已 generate · shortcut 不应 generate |
| 关闭页后下一位串台 | `e2e_exhibition_isolation` | release 未调 · sendBeacon 失败 |
| 改 wizard 后 edu 挂了 | `e2e_b_edu_batch` | 误改共享常量 · 破坏 `?mode=edu` 跳转 |

## 三、按错误类型修复路径

### 3.1 Launch / 环境（P0）

1. `echo $env:GODOT_PATH` 或读 `backend/.env`
2. 手动：`Godot.exe --path templates/{slug}` 是否 3s 内崩溃
3. `POST /play/launch` 看 `project_path` · `pid` · `play/status.running`
4. **允许改**：`backend/app/services/godot_launcher.py` · `backend/app/routers/play.py` · 环境说明文档  
5. **禁止**：为过关改 `templates/{slug}/core/**`

### 3.2 A 链 shortcut（P0/P1）

1. 确认 `SHORTCUT_FLOW = ["S0","S1","S9"]`
2. S1 提交后 `stepIndex` 是否跳到 S9（非 S2）
3. `launchGodotPlay` 是否在未 generate 时仍走 templates  
4. **允许改**：`kiosk/wizard.js` · `kiosk/index.html` · `kiosk/styles.css`  
5. 改后必跑：`e2e_exhibition_isolation.py`

### 3.3 跨链误伤（P0）

1. 跑 `e2e_b_edu_browser_smoke.py` 定位失败 slug
2. `git diff kiosk/edu/` 应为空（修复窗默认不改 edu）
3. 若 A 链改动导致 edu 失败 → **回滚 A 链** 或最小修复 `?mode=edu` 分流  
4. **禁止**：在修复窗大改 `edu-wizard.js` 主流程（属窗15/16 冻结）

### 3.4 Godot 运行时 ERROR（P0）

1. `godot-mcp run_project` 或 `Godot --quit-after 120`
2. 区分：**阻塞 ERROR**（SCRIPT ERROR / Parse Error）vs 警告  
3. 若仅 `config/game_config.json` tuning 可修 → 单窗单 slug · ±30%  
4. 若需 `core/` → **关闭修复窗**，开品类 R2 会话

### 3.5 遗留进程（P1）

窗17 批量 launch 可能留下 7 个 Godot 窗口。  
- 现场：任务管理器结束 `Godot*.exe`  
- 代码：确认 S9「完成体验」/ 页关闭 / `btnReset` 调 `release`；可选在 `launchGodotPlay` 提示「请先关闭上一窗口」
