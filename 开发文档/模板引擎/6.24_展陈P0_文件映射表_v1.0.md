## 二、E-P0-17 真代码链路

```text
POST /generate/v2 成功
    → workspace/{session_id}/config/game_config.json 落盘
    → GET /sessions/{id}/workspace/game-config  ✅
    → EduCodeViewer.setContent(content, highlightLines)
    → 禁止 getFallbackConfigSnippet() 冒充（有 workspace 时）
```

| 文件 | 职责 |
|------|------|
| `kiosk/edu/edu-wizard.js` | `getFallbackConfigSnippet` · `renderB6Ready` · `renderB7Play` |
| `kiosk/edu/code-viewer.js` | 左侧 JSON 展示与高亮行 |
| `backend/app/routers/sessions.py` | `GET .../workspace/game-config` 只读 endpoint |
| `backend/app/services/workspace_guard.py` | 路径越界拦截 |
