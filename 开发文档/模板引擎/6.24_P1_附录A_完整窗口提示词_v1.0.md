## A.1 窗1 · shmup

**模型**：Claude Sonnet（思考）或 Auto

```text
【6.24 P1 · 窗1 · shmup · 完整施工单】

仓库：E:\文三路AI馆\2.ai生成游戏
会话范围：仅 shmup · 仅新建/修改下列 4 文件

## 交付清单
1. config/creative_templates/shmup.json
2. config/code_anchors/shmup.json
3. assets/code_theater/shmup.jsonl
4. templates/_edu/shmup_hooks.gd

## 必读（先读后写）
- config/creative_templates/platformer.json
- config/code_anchors/platformer.json
- assets/code_theater/platformer.jsonl
- templates/_edu/platformer_hooks.gd
- templates/shmup/config/game_config.json
- templates/shmup/core/player_ship.gd（只读）
- config/optional_skills.json → shmup: bomb, laser_beam

## creative_templates/shmup.json
- genre: shmup · display_name: 街机飞机射击
- name_suggestions: 雷霆小队、星空守护者、小小飞行员、弹幕闪避
- clamp_percent: 30 · tuning_defaults_ref: templates/shmup/config/game_config.json
- 4 题：
  q_speed     → tuning.player.speed              默认300   慢210 / 默认300 / 快390
  q_fire_rate → tuning.player.bullet_interval_ms 默认200   稀260 / 默认200 / 密140（越小射越快）
  q_spawn     → tuning.spawn.interval_ms         默认1500  疏1950 / 默认1500 / 密1050（越小刷越密）
  q_skill     → skill_pick · tuning.enabled_skills · skill_ids:["bomb","laser_beam"] · optional:true
- 每 option 的 code_anchor_id 必须存在于 code_anchors

## code_anchors/shmup.json
| anchor / action_id | 绑定 | caption（儿童口语） |
| fire   | tuning.player.bullet_interval_ms · action_id:fire | 射得快慢在这里调！ |
| hit    | tuning.player.max_hp 或 core/player_ship.gd 受击 · action_id:hit | 被打到会很危险！ |
| pickup | tuning.powerup.duration_sec · action_id:pickup | 吃到道具会变强！ |
| player_speed · spawn_rate · enabled_skills | 填空用 · 无 action |
- line_hint 对照 game_config / player_ship.gd 真实行号

## code_theater/shmup.jsonl
- 约 50–70 行 · delay_ms 合计 ≈45000
- 高亮 speed / bullet_interval_ms / spawn.interval_ms · anchor_id 与 anchors 同源
- 末尾可摘录 player_ship.gd 射击片段

## shmup_hooks.gd
- extends Node · Godot 4.6 强类型 · 照 platformer_hooks 模式
- fire：_handle_auto_shoot 或子弹生成时上报
- hit：玩家受伤/无敌结束
- pickup：apply_powerup / powerup collected
- 桥不存在时静默 return

## 硬禁止
templates/shmup/core/** · backend/ · kiosk/ · edu_workspace.py · 其他 slug · 新建 PNG

## 验证
python 05-工具脚本/validate_creative_templates.py

## 收工
变更文件 · validate 输出 · core 无 diff · 题数/锚点/剧场行数/total_ms
```
