## 方向判定与 payload 样例

**判定**

```text
innerWidth < 900           → portrait
else orientation: portrait → portrait
else                       → landscape
```

**`getViewportPayload()` 示例**

```json
{
  "orientation": "portrait",
  "client_viewport": {
    "screen_x": 0,
    "screen_y": 0,
    "screen_w": 1080,
    "screen_h": 1920,
    "devicePixelRatio": 1,
    "kiosk_rect": { "x": 0, "y": 72, "w": 1080, "h": 1848 },
    "godot_zone_rect": { "x": 0, "y": 904, "w": 1080, "h": 1016 }
  }
}
```

（数值随 DevTools 视口与 header 高度略有浮动，属预期。）
