## 2. 方向检测（R2）

### 2.1 模块 `kiosk/edu/layout/orientation.js`

| API | 行为 |
|-----|------|
| `EduOrientation.mount()` | 监听 `resize` · `orientationchange` |
| `EduOrientation.getMode()` | 返回 `"landscape"` \| `"portrait"` |
| `EduOrientation.getViewportPayload()` | 返回 launch API 用的 `client_viewport` |

**判定逻辑（与代码一致）**

```text
若 window.innerWidth < 900 → portrait
否则若 matchMedia("(orientation: portrait)").matches → portrait
否则 → landscape
```

写入：`<body data-orientation="landscape|portrait">`

### 2.2 布局矩阵

| 模式 | 单栏步（B1–B2） | 双栏步（B3–B7） |
|------|----------------|----------------|
| **landscape** | 居中卡片 max-width 720px | 左栏 ≥45% 代码工作区 · 右栏预览/引导 |
| **portrait** | 全宽 padding 16px | **上行** 工作区/配方 · **下行** 预览/试玩引导 |

### 2.3 CSS 类名约定

| 选择器 | 含义 |
|--------|------|
| `body[data-orientation="landscape"] .dual-pane-root` | 两列 grid |
| `body[data-orientation="portrait"] .dual-pane-root` | 两行 grid |
| `.pane-right.godot-zone` | B6+ 试玩引导区强调边线（蓝虚线框示意游戏区） |

### 2.4 触控

- 竖屏下 `dual-toolbar` 固定底栏，不遮挡下行游戏区指引。
- 代码区 `touch-action: pan-y`（竖屏允许纵滚）。

## 3. 试玩面板与 Godot 窗口（R3）

### 3.1 浏览器内（kiosk）

| 阶段 | 横屏 | 竖屏 |
|------|------|------|
| B3–B4 右栏 | 品类 HTML 动画居中 | 同，占下行 |
| B6 左栏 | 真 `game_config` | 上行可滚动 |
| B6–B7 右/下栏 | `launch-status-card` + 操作说明 | 同，占下行 ≥50% 高 |
| B7 | 左代码高亮 + 右状态；演示钮底部 | 竖屏演示钮可收折 |

**不占浏览器 embed Godot**（与展陈 P0 一致）；右/下栏为 **引导 + 状态**，与外置窗配对。

### 3.2 外置 Godot 窗口（后端）

| 模式 | 目标矩形（主显示器） |
|------|---------------------|
| landscape | `x = screen_w * 0.5`, `y = 0`, `w = screen_w * 0.5`, `h = screen_h` |
| portrait | `x = 0`, `y = screen_h * 0.5`, `w = screen_w`, `h = screen_h * 0.5` |

**流程**

1. `edu-wizard.js` `launchGame()` 读取 `EduOrientation.getViewportPayload()`。
2. `POST /sessions/{id}/play/launch` body 含 `orientation` + `client_viewport`。
3. `godot_launcher.launch()` 启动后调用 `godot_window_layout.place_by_pid(pid, rect)`。
4. 失败：响应 `window_placed: false`，前端状态卡显示手动指引。

### 3.3 新增后端文件

- `backend/app/services/godot_window_layout.py` — Win32 `FindWindow` / `SetWindowPos`（按 PID 枚举子窗口）
- `backend/tests/test_godot_window_layout.py` — mock 或 skip 非 win32

### 3.4 `kiosk_edu_spec.layout` 扩展

```json
"layout": {
  "left_min_width_pct": 45,
  "orientation_breakpoint_px": 900,
  "godot_zone": {
    "landscape": { "anchor": "right", "width_pct": 50 },
    "portrait": { "anchor": "bottom", "height_pct": 50 }
  }
}
```
