# 需求文档

## 1. 应用概述

### 1.1 应用名称
街机飞机射击游戏\n
### 1.2 应用描述
一款街机风格的飞机射击游戏,玩家操控战机与多种类型敌机展开对战。游戏采用快节奏设计,配备完整音效系统,支持移动端触屏操作,为玩家带来紧张刺激的射击体验。
\n## 2. 用户与使用场景

### 2.1 目标用户
- 喜欢街机射击游戏的玩家
- 移动端休闲游戏用户
- 追求快节奏刺激体验的玩家

### 2.2 核心使用场景
- 碎片时间娱乐
- 挑战高分记录
- 体验街机射击乐趣

## 3. 页面结构与功能说明
\n### 3.1 页面结构

```\n├── 开始页面
├── 游戏主界面
└── 结束页面
```

### 3.2 功能说明

#### 3.2.1 开始页面
- 展示游戏标题
- 提供「开始游戏」按钮
- 点击按钮进入游戏主界面

#### 3.2.2 游戏主界面
\n**玩家飞机**
- 玩家飞机固定出现在屏幕下方区域
- 支持触屏拖动控制飞机左右移动
- 飞机自动持续发射子弹
- 被敌机子弹击中后游戏结束

**敌机系统**
- 敌机从屏幕上方随机位置生成
- 敌机向下移动并发射子弹攻击玩家
- 敌机包含多种类型,不同类型具有不同外观和行为模式:\n  - 普通敌机:匀速直线下降,固定频率发射单发子弹
  - 快速敌机:移动速度较快,发射频率较高\n  - 强化敌机:血量较高,需多次击中才能摧毁
- 敌机被玩家子弹击中后销毁并播放爆炸效果
- 敌机移出屏幕下方后自动消失

**武器系统**
- 玩家子弹和敌机子弹外观保持一致
- 子弹向目标方向直线飞行
- 子弹击中目标或移出屏幕后消失
\n**得分系统**
- 实时显示当前得分
- 击毁敌机获得分数
- 不同类型敌机分值不同

**音效系统**
- 玩家射击音效\n- 敌机爆炸音效
- 背景音乐循环播放
- 游戏结束音效

#### 3.2.3 结束页面
- 显示本局得分
- 提供「重新开始」按钮
- 点击按钮返回游戏主界面重新开始

## 4. 业务规则与逻辑
\n### 4.1 游戏玩法规则

**游戏目标**
- 操控飞机击毁尽可能多的敌机
- 避免被敌机子弹击中
- 获得更高分数

**游戏节奏**
- 快节奏设计,敌机生成频率适中
- 弹幕密度适中,保持紧张感但不过于密集
- 随游戏进行,敌机生成频率逐渐提升

**碰撞检测**
- 玩家子弹与敌机碰撞:敌机受到伤害或销毁
- 敌机子弹与玩家飞机碰撞:游戏结束
- 玩家飞机与敌机碰撞:游戏结束

### 4.2 敌机生成规则
- 游戏开始后持续生成敌机
- 生成位置随机分布在屏幕上方
- 不同类型敌机按一定比例随机生成
- 生成间隔时间随游戏进行逐渐缩短

### 4.3 得分规则
- 击毁普通敌机:10分
- 击毁快速敌机:20分
- 击毁强化敌机:30分\n\n### 4.4 游戏结束条件
- 玩家飞机被敌机子弹击中
- 玩家飞机与敌机发生碰撞
\n### 4.5 移动端触控操作
- 触摸屏幕任意位置并拖动:玩家飞机跟随手指移动
- 飞机移动范围限制在屏幕可视区域内
- 射击为自动持续进行,无需额外操作

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 玩家飞机移动到屏幕边缘 | 限制飞机不能移出屏幕范围 |
| 多个敌机同时被击中 | 分别计算得分并播放爆炸效果 |\n| 敌机子弹与玩家子弹碰撞 | 双方子弹均消失 |
| 音效加载失败 | 游戏正常运行,仅无音效 |
| 触控操作失灵 | 提示用户刷新页面重新进入 |

## 6. 验收标准

1. 进入开始页面,点击「开始游戏」按钮
2. 进入游戏主界面,玩家飞机出现在屏幕下方并自动发射子弹
3. 触摸屏幕拖动,玩家飞机跟随手指移动
4. 敌机从屏幕上方生成并向下移动,同时发射子弹\n5. 玩家子弹击中敌机,敌机销毁并播放爆炸音效,得分增加
6. 敌机子弹击中玩家飞机,游戏结束并播放结束音效
7. 进入结束页面,显示本局得分,点击「重新开始」返回游戏主界面\n
