# 需求文档

## 1. 应用概述

### 1.1 应用名称\nK12 无尽跑酷

### 1.2 应用描述
一款浏览器端横版跑酷小游戏，玩家控制角色自动向前奔跑，通过跳跃和滑铲躲避障碍物，在60秒内跑得越远得分越高，碰撞障碍后游戏结束并显示最终距离。

## 2. 用户与使用场景

### 2.1 目标用户
- K12学生群体
- 休闲游戏玩家
- 浏览器轻度游戏用户

### 2.2 核心使用场景
- 课间休息时快速游玩
- 碎片时间娱乐
- 挑战个人最高距离记录

## 3. 页面结构与功能说明

### 3.1 页面结构
\n```\n游戏应用
├── 开始页面
├── 游戏页面
└── 结束页面
```

### 3.2 功能说明\n
#### 3.2.1 开始页面\n- 展示游戏标题「K12 无尽跑酷」
- 显示开始按钮
- 点击开始按钮进入游戏页面

#### 3.2.2 游戏页面
- 角色自动向前奔跑\n- 显示倒计时（从60秒开始递减）
- 显示当前跑动距离
- 玩家操作：
  - 按空格键：角色跳跃
  - 按↓方向键：角色滑铲
- 障碍物生成：
  - 低栏障碍：需滑铲通过
  - 高栏障碍：需跳跃通过
- 背景滚动：明亮街景或森林侧视场景
- 碰撞检测：角色与障碍物发生碰撞时游戏结束
- 时间到达：倒计时归零时游戏结束
\n#### 3.2.3 结束页面
- 显示本次游戏跑动距离
- 显示重新开始按钮
- 点击重新开始按钮返回游戏页面重新开始

## 4. 业务规则与逻辑
\n### 4.1 游戏流程
- 开始页面 → 游戏页面 → 结束页面 → 游戏页面（循环）

### 4.2 跑动规则
- 角色以固定速度自动向前奔跑\n- 跑动距离持续累加并实时显示
\n### 4.3 操作规则
- 空格键触发跳跃动作，角色离地一定高度后落回地面
- ↓方向键触发滑铲动作，角色降低高度后恢复站立
- 跳跃和滑铲动作不可同时执行

### 4.4 障碍物规则
- 低栏障碍：高度较低，角色必须滑铲才能通过，跳跃或站立状态会碰撞
- 高栏障碍：高度较高，角色必须跳跃才能通过，站立或滑铲状态会碰撞
- 障碍物随机生成，保持合理间隔

### 4.5 碰撞规则
- 角色与任意障碍物发生碰撞，游戏立即结束
- 进入结束页面，显示本次跑动距离
\n### 4.6 计时规则
- 游戏开始时倒计时从60秒开始\n- 倒计时归零时游戏结束\n- 进入结束页面，显示本次跑动距离\n
### 4.7 计分规则
- 跑动距离越远，得分越高
- 距离以米为单位累加
\n## 5. 异常与边界情况
\n| 场景 | 处理方式 |
|------|----------|
| 玩家在跳跃过程中按↓键 | 忽略滑铲指令，等待跳跃动作完成 |
| 玩家在滑铲过程中按空格键 | 忽略跳跃指令，等待滑铲动作完成 |
| 倒计时归零时角色正在空中 | 游戏立即结束，不等待落地 |
| 碰撞发生时倒计时未归零 | 游戏立即结束，忽略剩余时间 |
| 障碍物生成间隔过短 | 确保玩家有足够反应时间，最小间隔不低于2秒 |
| 障碍物生成间隔过长 | 确保游戏节奏紧凑，最大间隔不超过5秒 |

## 6. 验收标准

1. 打开游戏，进入开始页面，点击开始按钮\n2. 进入游戏页面，角色自动向前奔跑，倒计时从60秒开始递减
3. 按空格键，角色跳跃并成功越过高栏障碍
4. 按↓方向键，角色滑铲并成功通过低栏障碍\n5. 角色与障碍物碰撞，游戏结束并显示跑动距离
6. 点击重新开始按钮，返回游戏页面重新开始游戏
