# 需求文档

## 1. 应用概述

### 1.1 应用名称\nK12 割草幸存者

### 1.2 应用描述
一款浏览器端的割草生存类小游戏，玩家通过WASD控制角色移动，自动攻击靠近的敌人，收集经验宝石升级强化能力，在180秒内存活即可获胜。游戏采用糖果卡通风格，适合K12年龄段用户。

## 2. 用户与使用场景

### 2.1 目标用户
- K12年龄段学生
- 休闲游戏玩家
- 割草生存类游戏爱好者\n
### 2.2 核心使用场景
- 课余休闲娱乐
- 碎片时间游戏体验
- 浏览器端快速启动游玩

## 3. 页面结构与功能说明

```\n游戏应用
├── 开始界面
├── 游戏主界面
│   ├── 游戏场景
│   ├── HUD显示层
│   └── 升级弹窗
└── 结束界面
```

### 3.1 开始界面
\n**功能说明**
- 展示游戏标题「K12 割草幸存者」
- 提供「开始游戏」按钮
- 点击按钮进入游戏主界面

### 3.2 游戏主界面

#### 3.2.1 游戏场景

**角色控制**
- 玩家使用WASD键控制角色移动
- W键向上移动\n- A键向左移动
- S键向下移动
- D键向右移动\n- 角色在场景内自由移动，不可移出屏幕边界

**自动攻击**
- 角色自动向距离最近的敌人发射攻击
- 攻击自动触发，无需玩家操作
- 攻击命中敌人造成伤害

**敌人生成**
- 敌人从屏幕外围环形区域生成
- 生成后向玩家角色位置移动靠近
- 敌人采用糖果/卡通怪物造型，无血腥元素

**经验宝石**
- 敌人被击败后掉落经验宝石
- 玩家角色接触宝石自动拾取
- 拾取宝石增加经验值

#### 3.2.2 HUD显示层
\n**计时器**
- 显示剩余游戏时间
- 格式为「剩余时间：XXX秒」
- 从180秒开始倒计时
- 倒计时归零时游戏结束

**经验条**
- 显示当前经验值进度
- 经验值累积至满时触发升级
\n#### 3.2.3 升级弹窗
\n**触发条件**
- 经验条累积满时弹出\n\n**升级选项**
- 同时展示3个升级选项供玩家选择
- 选项类型包括：
  - 攻击速度提升
  - 攻击伤害提升\n  - 移动速度提升
- 每次升级随机展示3个选项（可重复）

**选择机制**
- 玩家点击其中1个选项\n- 选择后弹窗关闭，游戏继续
- 选中的属性立即生效
- 经验条清空重新累积

### 3.3 结束界面

**胜利界面**
- 显示「胜利」提示
- 显示本局游戏时长「180秒」
- 提供「再来一局」按钮
\n**失败界面**\n- 显示「失败」提示
- 显示本局存活时长\n- 提供「再来一局」按钮\n
**按钮功能**
- 点击「再来一局」返回开始界面
\n## 4. 业务规则与逻辑\n
### 4.1 玩法规则

**游戏目标**
- 在180秒内存活即为胜利
- 角色生命值归零即为失败
\n**战斗机制**
- 角色自动向最近敌人发射攻击
- 攻击命中敌人扣除敌人生命值
- 敌人生命值归零时被击败
- 敌人接触角色时对角色造成伤害

**成长机制**
- 击败敌人掉落经验宝石\n- 拾取宝石累积经验值
- 经验值满触发升级
- 升级时从3个随机选项中选择1个强化

### 4.2 胜负条件

**胜利条件**
- 计时器倒计时归零（存活满180秒）
\n**失败条件**\n- 角色生命值归零\n
### 4.3 数值规则

**初始属性**
- 角色初始生命值\n- 角色初始攻击速度
- 角色初始攻击伤害
- 角色初始移动速度

**升级强化**
- 攻击速度提升：每次提升固定百分比
- 攻击伤害提升：每次提升固定数值
- 移动速度提升：每次提升固定百分比
\n**敌人属性**
- 敌人生命值
- 敌人移动速度
- 敌人接触伤害

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 角色移动至屏幕边界 | 阻止继续向边界外移动 |
| 同时有多个敌人距离相同 | 攻击其中任意一个 |
| 升级弹窗期间游戏状态 | 游戏暂停，敌人停止移动和攻击 |
| 经验条满时角色生命值归零 | 直接进入失败界面，不触发升级 |
| 倒计时归零时角色生命值归零 | 判定为胜利 |
| 多个经验宝石重叠 | 同时拾取所有重叠宝石 |\n
## 6. 验收标准

1. 进入开始界面，点击「开始游戏」按钮进入游戏主界面\n2. 使用WASD键控制角色移动，角色正常响应移动指令
3. 角色自动向靠近的敌人发射攻击，击败敌人后掉落经验宝石\n4. 拾取经验宝石累积经验值，经验条满时弹出升级弹窗
5. 在升级弹窗中选择1个强化选项，属性提升生效，弹窗关闭
6. 持续游玩直至计时器倒计时归零，进入胜利界面
7. 点击「再来一局」按钮返回开始界面
