# 需求文档

## 1. 应用概述

### 1.1 应用名称
马里奥风格横版闯关游戏
\n### 1.2 应用描述
一款经典横版卷轴闯关游戏，玩家控制角色在关卡中跑跳，踩踏敌人，收集金币，突破障碍，完成关卡挑战。游戏采用像素风格UI，提供流畅的物理效果和操作体验。

## 2. 用户与使用场景

### 2.1 目标用户
喜欢经典横版闯关游戏的玩家

### 2.2 核心使用场景
玩家通过键盘控制角色在关卡中移动、跳跃，踩踏敌人获得分数，收集金币，避开障碍物，到达终点完成关卡。

## 3. 页面结构与功能说明

### 3.1 页面结构

```\n马里奥风格横版闯关游戏
├── 开始界面
├── 游戏界面\n└── 游戏结束界面
```

### 3.2 开始界面

#### 3.2.1 界面元素
  - 游戏标题
  - 开始游戏按钮
  - 像素风格背景
\n#### 3.2.2 功能说明
  - 点击开始游戏按钮进入游戏界面
\n### 3.3 游戏界面

#### 3.3.1 界面元素
  - 游戏场景区域（横版卷轴视角）
  - 顶部信息栏
    - 当前分数
    - 剩余生命值
    - 当前关卡编号
    - 收集的金币数量
\n#### 3.3.2 游戏元素
  - 玩家角色\n  - 敌人
  - 砖块
  - 金币
  - 管道
  - 地面
  - 终点标志

#### 3.3.3 角色控制
  - 方向键左/A键：向左移动
  - 方向键右/D键：向右移动
  - 空格键：跳跃
\n#### 3.3.4 核心玩法
  - 角色在关卡中左右移动和跳跃
  - 踩踏敌人消灭敌人并获得分数
  - 收集金币增加分数
  - 碰撞砖块可能获得金币或道具
  - 管道作为障碍物或装饰元素
  - 到达终点标志完成关卡

#### 3.3.5 物理效果
  - 角色受重力影响自然下落
  - 跳跃具有抛物线轨迹
  - 角色与地面、砖块、管道发生碰撞检测
  - 角色与敌人发生碰撞检测

### 3.4 游戏结束界面

#### 3.4.1 界面元素
  - 游戏结果提示（通关成功/游戏失败）
  - 最终分数
  - 重新开始按钮
  - 返回开始界面按钮

#### 3.4.2 功能说明
  - 显示游戏结果和最终分数
  - 点击重新开始按钮重新进入游戏界面
  - 点击返回开始界面按钮返回开始界面

## 4. 业务规则与逻辑

### 4.1 生命值系统
  - 玩家初始生命值为3
  - 角色被敌人碰撞（非踩踏）扣除1点生命值
  - 角色掉落到关卡底部扣除1点生命值\n  - 生命值归零时游戏结束

### 4.2 分数系统
  - 踩踏敌人获得100分
  - 收集金币获得10分
  - 完成关卡获得500分
\n### 4.3 关卡系统
  - 游戏包含多个关卡
  - 完成当前关卡后进入下一关卡
  - 每个关卡难度递增
\n### 4.4 碰撞规则
  - 角色从上方踩踏敌人：敌人消失，角色获得分数并产生小幅弹跳
  - 角色从侧面或下方碰撞敌人：角色扣除生命值，敌人不消失
  - 角色碰撞砖块：阻挡移动，可能触发金币掉落
  - 角色碰撞管道：阻挡移动\n  - 角色碰撞金币：金币消失，角色获得分数
\n### 4.5 卷轴规则
  - 视角跟随角色水平移动
  - 角色位于屏幕中央偏左位置
  - 关卡场景向左滚动展示后续内容

### 4.6 胜负条件
  - 胜利条件：角色到达关卡终点标志
  - 失败条件：生命值归零

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 角色跳跃时碰到顶部砖块 | 停止上升，开始下落 |
| 角色在空中时按跳跃键 | 无效操作，不产生二段跳 |
| 角色移动到关卡左边界 | 无法继续向左移动 |
| 角色移动到关卡右边界 | 无法继续向右移动 |
| 角色掉落到关卡底部 | 扣除1点生命值，角色重置到关卡起点 |
| 生命值扣除后仍大于0 | 角色短暂无敌状态，继续游戏 |
| 完成最后一个关卡 | 显示通关成功界面 |

## 6. 验收标准

1. 进入开始界面，点击开始游戏按钮
2. 进入游戏界面，使用方向键/WASD控制角色左右移动
3. 按空格键控制角色跳跃，观察抛物线轨迹和重力效果
4. 踩踏敌人消灭敌人，分数增加100分
5. 收集金币，分数增加10分
6. 到达关卡终点标志，完成关卡，分数增加500分
7. 进入游戏结束界面，显示最终分数和游戏结果
