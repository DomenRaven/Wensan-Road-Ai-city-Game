# 需求文档

## 1. 应用概述

### 1.1 应用名称\nK12 欢乐赛车

### 1.2 应用描述
一款面向K12用户的浏览器竞速小游戏，采用俯视角卡通风格，玩家通过简单的键盘操作控制赛车在赛道上竞速，在90秒内完成尽可能多的圈数或到达终点。游戏参考《Horizon Chase》的arcade手感、《OutRun》的分段道路与加速感、《Micro Machines》的小场景竞速特点，提供轻松愉快的竞速体验。

## 2. 用户与使用场景

### 2.1 目标用户
- K12阶段学生（6-18岁）
- 喜欢休闲竞速游戏的用户
- 寻求短时间娱乐放松的用户

### 2.2 核心使用场景
- 课间休息时进行短时间游戏
- 家庭娱乐时间的亲子互动
- 碎片时间的快速娱乐

## 3. 页面结构与功能说明
\n### 3.1 页面结构

```\n├── 开始页面
├── 驾驶页面（游戏主界面）
└── 结算页面
```

### 3.2 功能说明\n
#### 3.2.1 开始页面\n- 展示游戏名称「K12 欢乐赛车」
- 展示游戏说明：操作方式（← → 转向，空格加速）、游戏时长（90秒）、目标（完成尽可能多圈数）
- 提供「开始游戏」按钮，点击后进入驾驶页面
\n#### 3.2.2 驾驶页面\n- **赛道显示**：俯视角展示赛道，赛道包含弯道、直道、障碍物\n- **赛车显示**：玩家控制的赛车以像素/卡通风格呈现，位于赛道上\n- **操作控制**：
  - 按下 ← 键：赛车向左转向
  - 按下 → 键：赛车向右转向
  - 按下空格键：赛车加速
- **计时显示**：屏幕顶部显示剩余时间，从90秒开始倒计时
- **圈数显示**：屏幕顶部显示当前完成圈数
- **碰撞反馈**：赛车撞到障碍物时速度降低，但不翻车，继续行驶
- **游戏结束**：倒计时归零后，自动跳转至结算页面

#### 3.2.3 结算页面
- 展示本局游戏结果：完成圈数、用时90秒
- 提供「再玩一次」按钮，点击后返回开始页面
\n## 4. 业务规则与逻辑\n
### 4.1 玩法规则
- 游戏时长固定为90秒
- 玩家通过键盘操作控制赛车在赛道上行驶
- 目标为在90秒内完成尽可能多的圈数或到达终点
- 赛车撞到障碍物时速度降低，但不会翻车或爆炸，可继续行驶
\n### 4.2 操作规则
- 仅支持3个按键操作：← 键、→ 键、空格键
- ← 键与 → 键控制赛车左右转向
- 空格键触发加速效果
\n### 4.3 计分规则
- 每完成一圈，圈数+1
- 游戏结束时，以完成圈数作为成绩
\n### 4.4 结束条件
- 倒计时归零时，游戏结束
\n### 4.5 视觉风格
- 赛车采用像素风格或卡通风格
- 赛道采用俯视角（顶视角）呈现
- 整体色彩明亮、轻松，符合K12用户审美

### 4.6 内容限制
- 禁止出现碰撞爆炸效果
- 禁止出现血腥暴力元素
- 禁止出现不适合K12用户的内容

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 赛车撞到障碍物 | 速度降低，赛车继续行驶 |
| 赛车驶出赛道边界 | 速度降低，赛车自动回到赛道内 |
| 倒计时归零 | 游戏结束，跳转至结算页面 |
| 玩家未完成任何圈数 | 结算页面显示圈数为0 |
| 同时按下多个操作键 | 优先响应转向键，空格键加速效果叠加 |

## 6. 验收标准

1. 打开游戏，进入开始页面，点击「开始游戏」按钮
2. 进入驾驶页面，使用 ← → 键控制赛车转向，使用空格键加速
3. 赛车在赛道上行驶，撞到障碍物时速度降低但继续行驶
4. 完成一圈后，圈数显示+1\n5. 倒计时归零后，自动跳转至结算页面
6. 结算页面显示完成圈数，点击「再玩一次」按钮返回开始页面
