# TPL-01 · 射击（FPS/TPS）· 设计规范与实现路径

> **slug**：`shooter` · **视角**：TPS 顶视 / FPS 简化 corridor · **D10**：L0 preset + L1 换 theme

---

## 一、品类概述

K12 展厅射击模板采用**简化单机射击**：TPS 为八向/四向顶视角移动 + 自动/半自动开火；FPS 为单走廊前进 + 准星瞄准。参考 Nex Machina、双摇杆射击手感规范，**禁止血腥与写实枪械**。

---

## 二、核心设计规则（行业共识）

### 2.1 核心循环

```
移动定位 → 开火命中 → 敌人消灭/波次推进 → 拾取/得分 → 下一波
```

### 2.2 控制与手感

| 规则 | 说明 | 参考 |
|------|------|------|
| 瞄准插值 | 方向输入**禁止 snap**，view 方向 lerp 向 target | GameDeveloper 双摇杆射击 |
| 输入缓冲 | 换弹/开火动画期间缓冲下一次输入 | 通用 action 游戏 |
| 屏幕包裹 | 顶视射击常用 wrap，降低挫败 | 品类规格 TPL-01 |
| 反馈四通道 | 命中闪白、击退、短 hit-stop、音效 | Nex Machina / VFX 可读性 |

### 2.3 战斗可读性（顶视）

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 预警 onset | 120–220ms | 敌人攻击前 telegraph |
| 冲击峰值 | 40–90ms | 命中闪白 |
| 单特效屏占比 | ≤12–18% | 防满屏噪声 |
| 无敌帧 | 600–800ms | 受击后 i-frames |

### 2.4 关卡/波次

- **消除 downtime**：刷怪间隔短、关卡紧凑（Nex Machina）
- **玩家置于危险区**：出生点可选屏幕中央增加紧张感
- **波次调度**：`interval_spawn` 或 `wave_top_down`，禁止动态改调度器类型

### 2.5 FPS 简化规则（K12）

- 单走廊、敌人从前方出现，**无爆头/血腥**
- 准星固定屏幕中心，鼠标/右摇杆转向
- 最多 3 键：移动、射击、换弹（换弹可省略 L0）

## 三、参数规格（对齐 game_config）

### core 层（锁定）

| 键 | 值 | 说明 |
|----|-----|------|
| `movement.mode` | `eight_direction` / `fps_corridor` | TPS/FPS 分支 |
| `movement.smooth_factor` | `0.15` | 位置插值 |
| `combat.fire_mode` | `auto_hold` | 按住连射 |
| `combat.hitbox_type` | `aabb` | 矩形命中 |
| `combat.damage_apply` | `on_overlap_once` | 同帧单次结算 |
| `combat.invincibility_ms` | `800` | 无敌帧 |
| `spawn.pattern` | `wave_top_down` | 波次入场 |

### tuning 层（AI 可改 ±30%）

| 键 | 默认 | 范围 |
|----|------|------|
| `player.speed` | 280 | 180–400 |
| `player.max_hp` | 3 | 1–5 |
| `bullet.speed` | 520 | 300–700 |
| `bullet.interval_ms` | 180 | 100–350 |
| `enemy.speed` | 120 | 60–220 |
| `wave.enemies_per_wave` | 8 | 4–15 |
| `tuning.camera_mode` | `tps` | `tps` \| `fps` |

## 四、Godot 4.6 实现路径（L0）

```
templates/shooter/
├── core/
│   ├── player_controller.gd      # CharacterBody2D / 3D 简化
│   ├── bullet_spawner.gd
│   ├── hit_detector.gd           # Area2D AABB
│   └── wave_manager.gd
├── config/game_config.json
└── demo_preset/
```

**步骤**：

1. TPS：`CharacterBody2D` + 八向 `Input.get_vector`，`velocity` 归一化
2. 子弹：`Area2D` 池化，`move_and_collide` 或 `position += dir * speed * delta`
3. 敌人：`wave_manager` 按 `spawn.pattern` 从屏外 Y 负方向生成
4. 命中：`area_entered` → 扣 HP → 闪白 shader 0.05s → 击退
5. FPS 分支：`camera_mode==fps` 时锁定 X 轴前进 + 左右 strafe

**Kenney 素材**：`assets/kenney/space/` 或 `top-down-shooter/`

## 五、K12 约束

- 敌人：机器人、气球、陨石等抽象目标
- 无血液粒子；受击用闪白 + 星星
- 操作 ≤3 键；单局 2–3 分钟
