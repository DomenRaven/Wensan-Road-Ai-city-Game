# TPL-06 · 横版闯关 · 设计规范与实现路径

> **slug**：`platformer` · **视角**：侧视 · **D10**：L0 + L1（首批模板）

---

## 一、品类概述

2D 平台跳跃：左右移动、跳跃、收集物、简单敌人。手感参考 **Celeste**：Coyote Time、Jump Buffer、可变高度跳。

---

## 二、核心设计规则

### 2.1 手感三件套（行业标准）

| 机制 | 推荐值 @60FPS | 说明 |
|------|---------------|------|
| Coyote Time | **100ms**（6 帧） | 离地后仍可跳 |
| Jump Buffer | **120ms**（7 帧） | 落地前预输入 |
| Variable Jump | release × **0.5** | 短按矮跳 |
| Fall Gravity | **1.7×** 上升重力 | 下落更快更利落 |

### 2.2 移动

| 参数 | Celeste 参考 | K12 默认 |
|------|--------------|----------|
| move_speed | ~220 u/s | 220 |
| jump_velocity | -105 (scale) | -380 px/s |
| acceleration | 1200 | 1200 |
| friction | 800 | 800 |
| air_control | 0.6 | 0.6 |

### 2.3 关卡设计

- 教学区：平地 → 小 gap → 需跳跃高度
- 检查点每 30–45 秒路程
- 单关 2–4 分钟，≤80 节点/场景

## 四、Godot 实现路径

```
templates/platformer/
├── core/
│   ├── player_platformer.gd    # CharacterBody2D
│   ├── coyote_jump.gd          # 可内联
│   ├── collectible.gd
│   └── patrol_enemy.gd
```

```gdscript
# 跳跃判定伪代码
var can_jump := is_on_floor() or coyote_timer <= COYOTE_TIME
var buffered := jump_buffer <= JUMP_BUFFER
if can_jump and buffered:
    velocity.y = jump_velocity
    jump_buffer = 999.0
```

1. `move_and_slide()` + 自定义 gravity
2. `TileMap` 关卡，`NavigationRegion2D` 可选
3. 相机 `Camera2D` position_smoothing = 0.08
