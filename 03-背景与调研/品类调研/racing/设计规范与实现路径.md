# TPL-10 · 赛车 · 设计规范与实现路径

> **slug**：`racing` · **视角**：顶视 / 伪 3D · **D10**：L0

## 一、品类概述

街机赛车：**顶视**（Micro Machines 风）或 **伪 3D**（OutRun 分段道路）。K12 L0 优先顶视；伪 3D 为可选 core 分支。

---

## 二、核心设计规则

### 2.1 街机 vs 拟真

| 街机 | K12 采用 |
|------|----------|
| 漂移保速 | 简化：松键自动回正 |
| 道具/武器 | L0 无 |
| 宽弯道 | 小弯道 +  forgiving 碰撞 |

### 2.2 顶视参数

```
steering = input_x * turn_rate * (speed / max_speed)  # 低速转向大
velocity = forward * speed
friction: speed *= 0.99 每帧（无油门时）
```

### 2.3 伪 3D（Lou's Pseudo 3D）

- 道路 = **segment 数组**，每段 `{curve, y, width}`
- 段线以**恒定速率**向玩家移动（非透视加速）
- 曲线力 + 玩家转向容量需迭代平衡（Horizon Chase）

### 2.4 核心循环

```
倒计时 → 沿圈/段行驶 → 计时/排名 → 完成
```

## 三、参数规格

### core 层

| 键 | 值 |
|----|-----|
| `render.mode` | `topdown` / `pseudo3d` |
| `physics.mode` | `arcade` |
| `collision.mode` | `slow_on_wall` |

### tuning 层

| 键 | 默认 | 范围 |
|----|------|------|
| `car.max_speed` | 320 | 220–420 |
| `car.acceleration` | 180 | 100–280 |
| `car.turn_rate` | 2.8 | 1.8–4.0 |
| `car.friction` | 0.96 | 0.92–0.99 |
| `track.laps` | 1 | 1–3 |
| `pseudo3d.segment_count` | 300 | 200–500 |

## 四、Godot 实现路径

**顶视 L0**：

```
templates/racing/
├── core/
│   ├── car_topdown.gd          # RigidBody2D 或 CharacterBody2D
│   ├── track_tilemap.gd
│   └── lap_counter.gd
```

**伪 3D（可选）**：

```
├── core/
│   ├── road_segments.gd        # 段数据 Resource
│   └── pseudo3d_renderer.gd    # 参考 extentofthejam
```

1. 顶视：`move_and_slide`，墙 `collision` 速度 ×0.5
2. 圈检测：checkpoint `Area2D` 序列
3. Kenney `racing-pack`
