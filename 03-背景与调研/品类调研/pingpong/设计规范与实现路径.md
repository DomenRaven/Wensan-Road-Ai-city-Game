# TPL-09 · 乒乓球 · 设计规范与实现路径

> **slug**：`pingpong` · **视角**：顶视球台 · **D10**：L0

---

## 一、品类概述

1v1 乒乓球/Pong 变体：左右移动球拍，球在台内反弹，**先达 11 分**胜（K12 可改为 5 分）。

---

## 二、核心设计规则

### 2.1 球拍反弹（经典 Pong 公式）

```
relative_y = (ball.y - paddle.center_y) / (paddle.height / 2)   # -1..1
bounce_angle = relative_y * MAX_ANGLE   # MAX_ANGLE ≈ 75° = 5π/12
ball_vx = speed * cos(bounce_angle) * sign
ball_vy = speed * -sin(bounce_angle)
```

- **球速恒定**或每 N 次击挡 +5%（上限 cap）
- 边缘击球 → 大角度；中心 → 水平

### 2.2 可选进阶

-  paddle 移动速度影响 spin（`jaridnft/pong-app` backspin）
- AI：追踪球 Y + 反应延迟 100–200ms

### 2.3 核心循环

```
发球 → 往返击球 → 出界/未接到 → 得分 → 11 分胜
```

## 三、参数规格

### core 层

| 键 | 值 |
|----|-----|

| `physics.speed_ramp` | `every_4_hits` |

| 键 | 默认 | 范围 |
|----|------|------|
| `ball.base_speed` | 400 | 280–550 |
| `ball.max_speed` | 650 | 500–800 |
| `paddle.speed` | 350 | 250–450 |
| `paddle.height` | 120 | 80–160 |
| `ai.reaction_ms` | 150 | 80–250 |

## 四、Godot 实现路径

```
templates/pingpong/
├── core/
│   ├── ball.gd                 # CharacterBody2D 或 kinematic
│   ├── paddle.gd
│   ├── score_manager.gd
│   └── ai_paddle.gd
```

1. 球 `move_and_collide`；碰撞 paddle 时按上式改 `velocity`
2. 墙/台边：`velocity.bounce(normal)`
3. 出界检测：`Area2D` 超出台 → 对方得分
4. 顶视相机固定
