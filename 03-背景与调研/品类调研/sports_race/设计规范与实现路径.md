# TPL-08 · 体育竞速 · 设计规范与实现路径

> **slug**：`sports_race` · **视角**：侧视跑道 · **D10**：L0

---

## 一、品类概述

横向**计时赛跑**：自动或半自动前进，跳跃越障，单键加速。与 `parkour` 区别：强调**圈速/计时**与跑道 UI，障碍较稀疏。

---

## 二、核心设计规则

### 2.1 与跑酷差异

| 维度 | parkour | sports_race |
|------|---------|-------------|
| 目标 | 距离生存 | **最快完赛时间** |
| 操作 | 跳/换道 | 跳 + **加速** |
| 障碍 | 密集 | 稀疏、节奏型 |
| 反馈 | 距离 HUD | **计时器 + 名次** |

### 2.2 核心循环

```
起跑 → 奔跑/加速 → 越障 → 冲线 → 显示时间 → 重试
```

### 2.3 手感

- 继承 platformer：`coyote_time` + `jump_buffer`
- 加速：按住键 `speed = lerp(speed, max, accel * delta)`
- 障碍：`distance_based` 固定 pattern（每 200px 一个）

## 三、参数规格

### core 层

| 键 | 值 |
|----|-----|
| `movement.mode` | `auto_run_forward` |
| `movement.lane_count` | `1` |
| `score.mode` | `distance_time` |
| `obstacle.spawn_rule` | `distance_based` |

### tuning 层

| 键 | 默认 | 范围 |
|----|------|------|
| `runner.speed` | 240 | 160–360 |
| `runner.boost_multiplier` | 1.4 | 1.2–1.8 |
| `jump.velocity` | -420 | -300 ~ -520 |
| `track.length_px` | 3000 | 2000–5000 |
| `round.duration_sec` | 60 | 45–90 |
